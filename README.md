# PROYECTO-MC-Astronomia-Computacional-2
Monte Carlo simulation for AST-305, modeling the Milky Way's stellar population. The code generates N stars with a Salpeter IMF and constant SFR. It applies stellar evolution (T∝M⁻²·⁵) and IFMRs from Kalirai (2008) &amp; Raithel (2018) to estimate the final fractions of White Dwarfs (WDs), Neutron Stars (NSs), and Black Holes (BHs).
